# @Yujia Activation — East Meets Feed

## Project Overview

This is a NYU Social Media & Brand course project documenting the **East Meets Feed** social media activation campaign for Yujia. The project includes presentation slides, a performance report, and supporting campaign materials.

## File Structure

```
@Yujia Activation/
├── @Yujia presentation final ver.2.html   # Main presentation deck
├── East_Meets_Feed_Outline.html            # Slide outline document
├── performance_report.html                 # Campaign performance report
├── slides_1_2_3.html                       # Partial slides export
├── East_Meets_Feed_Performance_Report.pdf  # PDF version of report
├── East_Meets_Feed_Script.pdf              # Campaign script PDF
├── images/                                 # Image assets for HTML files
│   ├── img_01_cover.png ... img_21_dy_hero.png
│   └── (activation screenshots)
├── 5:28 IG after activation/               # Instagram post-activation screenshots
├── 5:28 Red Note after activation/         # Xiaohongshu post-activation screenshots
├── 5:28 Tiktok after activation/           # TikTok post-activation screenshots
├── 5:31 IG after activation/               # Instagram follow-up screenshots
├── 5:31 Tiktok after activation/           # TikTok follow-up screenshots
├── Social numers before activation/        # Baseline metrics screenshots
└── Yujia pdf/                              # Source PDF slides (3 parts)
```

## Design System

All HTML files share a consistent visual identity:
- **Font:** Inter (Google Fonts)
- **Brand color:** `#c9a96e` (gold/warm accent)
- **Dark background:** `#1a1a1a`
- **Light background:** `#f5f4f0` or `#fff`
- **Slide canvas size:** 1080×607px (16:9)

## Platforms Covered

The activation campaign spans three platforms:
- **Instagram (IG)**
- **TikTok / Douyin**
- **Xiaohongshu / Red Note (XHS)**

## Notes

- HTML files are self-contained presentation slides styled for browser/print export (1080px wide)
- Screenshots in dated folders track social metrics before and after the activation (May 28 & 31, 2026)
- The `images/` directory contains optimized (`_sm`) and full-size versions of assets
